# 🔮 数学可视化 (Math Visualizer)

> 基于 Three.js 和 AI 的 3D 数学可视化工具 — 让抽象的数学概念变得直观可见

## ✨ 功能特性

- **AI 驱动生成** — 输入数学问题描述，AI（智谱 GLM-4-Flash）自动生成 3D 可视化场景
- **3D 交互** — 支持鼠标拖拽旋转、滚轮缩放、右键平移视角
- **立体几何展示** — 正方体、圆柱、球体、圆锥等几何体，带公式标注和参数显示
- **应用题解析** — 水池注水、体积计算等实际数学问题可视化
- **函数曲面** — 3D 数学函数图像（如 z = x² + y² 旋转抛物面）

## 🚀 快速开始

1. 克隆仓库后，直接用浏览器打开 `index.html`
2. 点击左侧「API 设置」配置智谱AI API Key（[免费注册获取](https://open.bigmodel.cn/)）
3. 在底部输入框描述数学问题，或点击右侧快速示例
4. 按 Enter 生成 3D 场景

## ⌨️ 快捷键

| 按键 | 功能       |
|------|------------|
| `R`  | 重置视角   |
| `T`  | 俯视图     |
| `F`  | 前视图     |
| `C`  | 清空场景   |
| `1-6`| 加载示例   |

## 🛠️ 技术栈

- **Three.js** — 3D 渲染引擎
- **智谱AI GLM-4-Flash** — AI 场景生成
- 纯前端，无需构建工具

## 📸 示例

- 正方体体积 V=a³
- 圆柱体 V=πr²h
- 球体表面积 S=4πr²
- 圆锥体 V=⅓πr²h
- 长方体水池注水问题
- 旋转抛物面 z=x²+y²
